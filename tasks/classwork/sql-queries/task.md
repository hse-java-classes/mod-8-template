# Практика: SQL-запросы на sql-ex.ru

**Тег коммита:** `sql-queries`
**Баллы:** 3

---

## Описание

Решите задачи на платформе [sql-ex.ru](https://sql-ex.ru/learn_exercises.php) (раздел «Учебные упражнения») и сохраните решения в репозиторий. Контекст — учебная БД с таблицами `Product`, `PC`, `Laptop`, `Printer`.

## Требования

Обязательно решить и сохранить четыре задачи:

| Файл | Задача на sql-ex.ru |
|------|---------------------|
| `task_02.sql` | Задача 2 |
| `task_14.sql` | Задача 14 |
| `task_05.sql` | Задача 5 |
| `task_12.sql` | Задача 12 |

Дополнительно: одна любая задача с 1 по 20 (кроме уже перечисленных) — файл `task_NN.sql`.

В каждом файле: первая строка — комментарий `-- Задача N: кратко суть`, далее один принятый платформой SQL-запрос.

Файлы кладите в каталог `sql/` в корне репозитория; имя `task_NN.sql` с двузначным номером (`task_02.sql`, не `task_2.sql`). Заглушки, неработающий SQL и списывание чужих решений не допускаются.
